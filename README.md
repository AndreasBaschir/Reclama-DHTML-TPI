# Aplicație Web - Casetă de Reclamă Dinamică

## Descriere

Aplicația este o pagină web simplă care implementează o casetă de reclamă interactivă folosind **DHTML (Dynamic HTML)**, respectând exact cerința inițială.

## Cerințe îndeplinite

- ✅ **Afișare automată** - Reclama apare la fiecare încărcare a paginii
- ✅ **Durată 10 secunde** - Se afișează pe ecran exact 10 secunde
- ✅ **Aruncată în față** - Caseta apare centrat pe ecran cu z-index ridicat
- ✅ **DHTML** - Implementare cu HTML5 + CSS3 + JavaScript vanilla

## Funcționalități suplimentare

- Overlay semi-transparent pe fundal pentru focus
- Animații fluide de apariție și dispariție
- Countdown timer vizibil (10, 9, 8... secunde)
- Buton pentru închidere manuală
- Buton pentru redeschidere reclamei
- Design modern cu gradient și shadow effects

## Structură fișiere

```
toy_project/
├── index.html      # Structura semantică
├── styles.css      # Design și animații CSS3
├── script.js       # Logica JavaScript pentru comportament dinamic
└── README.md       # Documentația proiectului
```

## Tehnologii folosite

- **HTML5** - Structură semantică
- **CSS3** - Design, animații keyframes (slideIn, slideOut, fadeIn)
- **JavaScript Vanilla** - Event listeners, DOM manipulation, timers
- **DHTML** - Interactivitate și dinamică

## Cum se folosește

1. Deschideți fișierul `index.html` într-un browser web
2. Reclama va apărea automat la încărcare
3. Se va afișa timp de 10 secunde
4. Puteți:
   - Așteptați ca să dispară automat
   - Apăsați butonul "Închide reclama" pentru a o închide manual
   - Apăsați "Deschide reclama din nou" pentru a o redeschide

## Descriere detaliată

Aplicația respectă pe deplin cerința academică de a crea o pagină web cu reclamă care:
- Apare automat la fiecare încărcare
- Dispare după 10 secunde
- Utilizează tehnologiile DHTML (Dynamic HTML)
